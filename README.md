# ![Bit Wave!](arts/logo-full.svg)

A simple media player that wrappers MPV.

## Features

- [x] Play local media files
- [ ] Play online media files
- [x] Media metadata
- [ ] Music lyrics
- [ ] Library and playlist management
- [ ] Various fine-tuning and control of the player
